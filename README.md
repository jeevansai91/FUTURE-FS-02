Full-Stack Lead Management System
"Where Opportunities become Outcomes"
LeadSphere is a robust, full-stack CRM (Customer Relationship Management) application built to bridge the gap between initial lead acquisition and final customer conversion. This project was developed as a capstone submission for my Web Development Internship.

🌟 Key Features
🔐 Secure Authentication: Protected routes for authorized personnel only.
📊 Visual Sales Analytics: Integrated Recharts Pie Chart to visualize the sales pipeline distribution (New vs. Converted) in real-time.
⚡ Lead Lifecycle Management (CRUD): Complete ability to Create, Read, Update, and Delete client records with interaction notes.
🔍 Optimized Search: A high-performance filtering system to find specific leads by name or email instantly.
📱 Responsive Design: Fully optimized for desktop and mobile viewing.

🛠️ Tech Stack & Architecture
Frontend: React.js (Functional Components & Hooks)
Backend: Node.js & Express.js (RESTful API Design)
Database: MongoDB Atlas (Cloud NoSQL Database)
State Management: React useState and useEffect for seamless data flow.
API Handling: Axios for secure frontend-to-backend communication.

📂 Project Structure
To ensure high maintainability, the project follows a clean separation of concerns:
/client: Contains the React frontend and UI components.
/server: Contains the Express API, Database models, and Controller logic.
